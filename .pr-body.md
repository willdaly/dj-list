## Summary

Integrates a music knowledge graph into the DJ List app, adding intelligent track discovery features powered by embeddings trained on the vinyl collection.

### What's included

**Knowledge Graph Pipeline** (Python)
- Training script using PyKEEN (TransE embeddings, 200 epochs)
- MongoDB enrichment script that maps embeddings back to song documents
- Adds camelotCode, harmonicCodes, energyTier, setCategory, and similarSongIds to every song

**Backend API**
- findHarmonicMatches — find songs compatible via the Camelot wheel
- findSimilar — cosine-similarity search across KG embeddings
- findNextTracks — DJ-friendly next-track suggestions (harmonic + energy aware)
- findByCamelot / findByEnergyTier — filter songs by Camelot code or energy tier with optional genre cross-filters

**Frontend UI**
- Harmonic Matches, Similar Tracks, and Next Track buttons on selected songs
- Camelot Code and Energy Tier columns in the results table
- Camelot and Energy dropdown filters with search buttons
- Back-to-results navigation for KG drill-down views

**Tests**
- Playwright integration tests for harmonic matches, similar tracks, and next-track suggestions
